Collaboration

The collaboration project provides a small frame work which enables you to apply your multiple core CPU's to partitionable tasks. A partition is a subset of your input data which can be processed from other partitions. Therefor by applying multiple CPU's in parallel processing independent partitions a speed up of processing time can be achieved.

Some examples can be found here
